Package Creation Guide

Step-by-Step:

  1. Select Distributor: Choose who will receive this package
  2. Add Items: Select items and enter requested quantities
  3. Allocate Stock: View stock at each depot and allocate quantities:
    • Click Auto-fill to use depot with most stock
    • Click depot button to fill from that location
    • Or manually enter quantities from multiple depots
  4. Monitor Progress: Watch the progress bar - green means fully allocated
  5. Submit: Create the package (partial allocations allowed)
Tip: Total allocated cannot exceed requested quantity. System prevents over-allocation.
